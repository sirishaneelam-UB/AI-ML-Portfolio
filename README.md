# Sirisha Neelam — AI/ML Product Portfolio

A professional portfolio that presents AI/ML coursework through a product-management and change-leadership lens.

## Featured artifacts

- **Spotlight Speakers:** a seven-person, 0→1 AI Lab that turns detailed speaker interviews into an AI-assisted booking experience.
- **AI/ML Timeline:** a concise history of major shifts in artificial intelligence and their implications for leaders.
- **Responsible AI Practice:** a six-part framework synthesized from course learning and assessments.

The site also includes redesigned, accessible process and marketing infographics for Spotlight Speakers.

## Local development

```bash
pnpm install
pnpm dev
pnpm build
```

Built with Next.js, React, and vinext for Cloudflare-compatible deployment.
